Quick Start:
1. Clone the project
	- run ```git clone the_link_to_the_repo```
2. Install Backend Dependencies
	- Go to base directory
	- run ```yarn install```
3. Install Frontend Dependencies
	- Go into client directory by running ```cd client```
	- run ```yarn install```
4. Start the project
	- run ```yarn start```
	- Go to localhost:3000 for the homepage
